# Change Docs Folder

Semua perubahan kode/fitur wajib punya dokumen di folder ini.

Format nama file:
- `CHG-YYYYMMDD-<slug-singkat>.md`

Contoh:
- `CHG-20260215-sync-push-validation.md`
- `CHG-20260216-wa-template-renderer.md`

Template:
- Gunakan `docs/CHANGE_DOC_TEMPLATE.md`.

Aturan:
1. Buat dokumen sebelum coding (status `planned`).
2. Update selama implementasi (status `in_progress`).
3. Tutup dengan bukti validasi (status `done`).
